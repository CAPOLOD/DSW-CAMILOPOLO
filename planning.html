Contenido de lectura

HTML no es un lenguaje de programación; es un lenguaje de marcado que define la estructura de tu contenido. HTML consiste en una serie de elementos que usarás para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera. Las etiquetas de encierre pueden hacer de una palabra o una imagen un hipervínculo a otro sitio, se pueden cambiar palabras a cursiva, agrandar o achicar la letra

Actualmente la nueva versión de este es la 5, Estos nuevos contenidos de HTML (HTML 5) vienen a mejorar distinto problemas que dejaba la estructura del HTML 4 al crear un documento, el HTML 4 y su algoritmo de esquematizado es muy tosco y genera numerosos problemas.Por ende hablaremos de estas nuevas etiquetas que vienen a facilitarnos más la vida a la hora de ingresar códigos.
